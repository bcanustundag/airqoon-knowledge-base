# Bursa Büyükşehir Belediyesi

**Bursa Büyükşehir Belediyesi** (Bursa Metropolitan Municipality) is a key municipal client for [[wiki/entities/airqoon|Airqoon]]. 

## Airqoon Implementation
Bursa utilizes Airqoon's solutions for a comprehensive, integrated air quality monitoring network spanning both the metropolitan center and surrounding districts. 

Notable use cases in Bursa include:
- Standard urban air quality monitoring.
- **Forest fire tracking/early warning.**
- Urban noise measurement.
- Public transparency via a dedicated citizen map interface (`map.airqoon.com/bursa`).
