# Sniffer4D

**Sniffer4D** is a state-of-the-art modular air quality mapping system specifically designed for integration with Unmanned Aerial Systems (UAS), primarily DJI enterprise drones. It is a key reference point in the [[wiki/concepts/drone-based-monitoring|drone-based monitoring]] landscape.

## Key Features
- **Capabilities:** Can measure up to 9 gases simultaneously (VOCs, SO₂, NO₂, O₃, CO, NH₃) alongside particulate matter.
- **Hardware Architecture:** Utilizes an active intake system drawing approximately 5 L/min. Each sensing module is equipped with a dedicated 16 MHz microcomputer to execute proprietary filtering and bias correction algorithms locally.
- **Data Quality:** Distributed processing and robust electromagnetic shielding enable high-accuracy readings (R-squared 0.81–0.95) and ppb-level resolution, mitigating the intense noise generated by drone motors.
- **Data Management:** Offers real-time 3D pollution mapping transmitted via LoRa or 4G.
