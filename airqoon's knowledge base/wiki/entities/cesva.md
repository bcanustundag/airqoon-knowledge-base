# Cesva

## Overview
Cesva (founded 1969, Barcelona) manufactures professional sound measurement instruments. They participate in national and international standards committees (IEC 61672-1, IEC 60942) and distribute to 40+ countries.

## Key Product: TA150 Noise Sensor
- **Class 1** precision (IEC 61672-1)
- 1/3-octave spectrum analysis (6.3 Hz – 20 kHz) with optional FR150 module
- Audio recording capability (MP3, FLAC, WAV) with optional GA150 module
- Measurement range: 25.5–137 dBA
- Connectivity: Ethernet, Wi-Fi, 4G+GPS (MR154 module)
- External outputs: RS-232, RS-485 (Modbus RTU), 4-20 mA current loop
- Internet protocols: HTTP/HTTPS, MQTT, UltraLight 2.0, Generic JSON, Sentilo API
- Power: Mains, PoE, 5-12 VDC, internal lithium battery (BA150: 24 h, BA151: 5 days)
- IP65 rated, mini-OLED status screen, volatile memory ~2 months
