# Denizli Büyükşehir Belediyesi

**Denizli Büyükşehir Belediyesi** (Denizli Metropolitan Municipality) is a municipal client of [[wiki/entities/airqoon|Airqoon]].

## Airqoon Implementation
In Denizli, the primary focus of the Airqoon environmental monitoring network is tracking traffic-induced air and noise pollution. The municipality provides data transparency to its citizens via a dedicated map interface (`map.airqoon.com/denizli`).
