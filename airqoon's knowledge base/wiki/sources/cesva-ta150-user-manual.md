# Cesva TA150 User Manual

## Summary
The user guide for the Cesva TA150 noise sensor. Provides instructions on device installation, setup, web server configuration, power supply options, data transmission over Ethernet/Wi-Fi/4G, and external output configurations (current loop, RS-232, RS-485). Also details maintenance procedures and acoustic verification.
