# Case Study: Mining air pollution monitoring network

## Summary
Gecelca S.A. E.S.P. created an air pollution monitoring system at Las Palmeras coal mine in Colombia, using Kunak AIR Pro to measure gases and PMs alongside Grimm EDM180.
