# Source: Airqoon Catalog Preliminary v5.2

**Raw file:** `raw/articles/airqoon-catalog-preliminary-v5.2.pdf`

## Summary

The Airqoon Product Catalog (Preliminary v5.2, English) outlines the full hardware lineup, including Unit L (outdoor), Unit M (indoor), Unit N (noise), and external addons.

## Key Facts Extracted

### Unit L (Outdoor)
- **Role:** Cost-effective and easy-to-use outdoor sensor for PM and gas pollutants.
- **Specs:** 3.2kg with solar panel, IP34 enclosure, 15.2Ah battery, 2G/4G/NB-IoT/WiFi. Operating temp: -10°C to +65°C.

### Unit M (Indoor)
- **Role:** Portable sensor system for indoor public spaces.
- **Specs:** <1kg, dimensions 12x12x6cm, IP34, 5V DC, 7.6Ah battery.
- **Measurements:** PM1, PM2.5, PM4, PM10, CO2 (NDIR), tVOC (MOx).
- **Connectivity:** WiFi, BLE, 4G, NB-IoT.

### Unit N (Noise)
- **Role:** Network-based noise sensor for urban and industrial areas.
- **Specs:** 30-130 dB range, Type 2 IEC 61672 standard (±3dB). RS485 output, IP65 enclosure. Power consumption <20mW.

### Addons
- Noise sensor, Wind Speed/Direction (Anemometer), Solar Panel, 5V Adapter, Extended Solar Power Package (ESP).
