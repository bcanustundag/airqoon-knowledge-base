# Cesva TA150 Datasheet

## Summary
Datasheet for the Cesva TA150 spectrum analyzer sensor for noise measurement (1/3 octave). It features Class 1 precision according to IEC 61672-1, multiple time and frequency weightings, external connectivity via PoE, mains, battery, Wi-Fi, Ethernet, 4G, and integration capabilities via Modbus, MQTT, Sentilo, and JSON over various hardware interfaces.
