# Airqoon vs Oizom Comprehensive Comparison

## Introduction

This document provides a detailed comparison between two leading air quality monitoring solutions: the Airqoon Unit L and the Oizom Polludrone. Both systems offer real-time environmental monitoring capabilities but target different market segments and use cases.

## Key Similarities

### 1. **Multi-Parameter Monitoring**

- Both devices measure particulate matter (PM) and multiple gas concentrations
- Essential meteorological parameters (temperature, humidity, pressure) included in both systems
- Noise monitoring available as additional module/unit
### 2. **Real-Time Data Monitoring**

- 24/7 continuous measurement with instant cloud data transmission
- Configurable data collection intervals (minute-based, 5-minute intervals, etc.)
### 3. **Cloud-Based Analytics and Visualization**

- Proprietary cloud platforms: Airqoon Lens and Oizom Envizom
- Historical data analysis, charts, maps, and automated reporting
- API integration for third-party system data transfer
### 4. **Solar Power Operation**

- Both designed for solar energy operation with adapter backup
- Battery support ensures uninterrupted monitoring during power outages
### 5. **Modular Design with Optional Sensors**

- Additional sensors/modules can be added to base measurement parameters
- Customizable based on specific project requirements

## Cloud Platform Comparison

- **Compare Mode** for side-by-side multi-location comparison
- **48-hour and inter-period change charts** to reveal patterns and anomalies
- Airqoon Lens offers **flexible data export** for seamless integration with your analysis tools
- Airqoon Lens provides** ****GenAI-backed report analysis** as an executive summary in a monthly basis
- Airqoon can provide** ****on-premise cloud solution** depending on the customer requirement

**Airqoon Public Map** enables active engagement with local authorities and communities by collecting citizen complaints, providing wildfire maps, and issuing emergency notifications.
